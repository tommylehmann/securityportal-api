Deployment note
---------------
This bundle is embedded at compile time via //go:embed static/redoc.standalone.js
(pkg/web/openapi.go) and served at GET /api/redoc.standalone.js — same origin,
no CDN (C-37/SA-54). The file must be committed to the repository so the Go
embed directive can include it in the binary; it is NOT generated at build time.
